# Настройка Telegram уведомлений

## 1. Создание Telegram бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям, введите название и имя пользователя для бота
4. Сохраните полученный **Bot Token** (примерно так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## 2. Получение Chat ID

### Метод 1: Через @userinfobot
1. Найдите бота [@userinfobot](https://t.me/userinfobot)
2. Отправьте ему команду `/start`
3. Скопируйте ваш **Chat ID** (число, например: `123456789`)

### Метод 2: Через API
1. Отправьте любое сообщение вашему боту
2. Перейдите по ссылке: `https://api.telegram.org/bot<BOT_TOKEN>/getUpdates`
3. Найдите `"chat":{"id":ВАШЕ_ID}` в ответе

## 3. Настройка переменных окружения

Создайте файл `.env.local` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

## 4. Обновление .gitignore

Убедитесь, что `.env.local` добавлен в `.gitignore`:

```
.env.local
```

## 5. Тестирование

После настройки переменных окружения и перезапуска сервера разработки, при отправке формы waitlist вы будете получать уведомления в Telegram.

## Формат сообщения

```
🎯 Новая заявка на waitlist!

👤 Имя: Иван Иванов
📧 Email: ivan@example.com
💬 Мессенджер: @ivan_telegram

📊 Осталось мест: 87/500
```

## Безопасность

- **Никогда не коммитьте** `.env.local` в репозиторий
- Bot Token и Chat ID должны оставаться конфиденциальными
- В продакшене используйте переменные окружения хостинг-провайдера (Vercel, Railway, etc.)
